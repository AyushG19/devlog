# DevLog

## Demo

<video src="assets/demo.mp4" controls width="100%" height="auto"></video>

A personal development logging tool to track progress, ideas, and milestones in your projects. Keep your dev journey organized and shareable!

## Features

- Log daily entries with timestamps
- Categorize logs by project or topic
- Search and filter past entries
- Interact with other users that are on thir journey as well

## Tech Stack

- **Frontend**: React, TaileindCSS, HTML5, CSS3, Vanilla JavaScript (for simplicity and performance)
- **Backend**: Node.js with Express (lightweight API server)
- **Database**: postgres (easy setup, no external dependencies)
- **Deployment**: Vercel or Netlify (static hosting with serverless functions)

## Quick Start

1. Clone the repo:
   ```
   git clone https://github.com/AyushG19/devlog.git
   cd devlog
   ```
2. Install dependencies:
   ```
   npm install
   ```
3. Run locally:
   ```
   npm start
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser.
